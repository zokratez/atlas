# Cross-Post Workflow

## Rule

Cross-posting is allowed only after re-exporting a clean, watermark-free version for each platform.

## Workflow

1. Produce the TikTok-first cut.
2. Export a clean source file with no platform watermark.
3. Upload natively to TikTok.
4. Wait 60-90 minutes.
5. Upload natively to Instagram Reels with platform-native caption edits.
6. Wait 60-90 minutes.
7. Upload natively to YouTube Shorts with search-friendly title and description.
8. Record performance notes in `vault/04-memory/learning-log.md`.

## Platform Notes

TikTok is the discovery engine. Shorts compounds through search. Reels supports iOS-native discovery and emotional/family angles.
